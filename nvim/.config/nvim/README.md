# NeoVim Configuration

## Commands
| Command        | Description                      |
| -------------- | -------------------------------- |
| `:Lazy`        | Open Lazy plugin manager         |
| `:Mason`       | Open LSP Manager                 |


## Keybindings

| Bind          | Description       |
| ------------- | ----------------- |
| `CTRL + n`    | Open Filetree     |
| `SHIFT + H`   | Show hidden files |
| `CTRL + p`    | Find files        |
| `SPACE + fg`  | Grep string       |
| `SPACE + ca`  | Open code actions |
| `SPACE + ff`  | Format file       |
